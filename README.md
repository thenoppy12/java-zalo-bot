**<p align="center">do rảnh rỗi sinh nông nỗi nên mới thử tay port cái này qua java hehe<br>Inspired of the official [python-zalo-api](https://pypi.org/project/python-zalo-bot)<br>(này làm chơi cho vui thôi chớ không có ý định lâu dài)</p>**

## Usage
```gradle
# build.gradle
repositories {
    maven {
        name = "FkrystalMaven"
        url = uri("https://maven.fkrystal.qzz.io/releases")
    }
}
ZALOAPI_VERSION = "find it on my repo."
dependencies {
    implementation("neuralseal.fkrystal:zalo-api:${ZALOAPI_VERSION}")
}
```
**for code usage, please read the [Test](https://github.com/thenoppy12/java-zalo-bot/blob/master/src/test/java/Test.java) class**
